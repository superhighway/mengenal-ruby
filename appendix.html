Install Ruby Multiversi

Biasanya kalau sudah ada project besar dan mau upgrade, perlu coba-coba sendiri dulu bagaimana versi baru mempengaruhi code project yang sudah ada. Karena itulah, install Ruby Multiversi diperlukan.

Untuk mengatur instalasi Ruby multiversi, kamu bisa menggunakan RVM atau rbenv. Selama ini saya masih pake RVM, jadi sejujurnya masih kurang tahu rbenv seperti apa. Kalau kamu mau pakai RVM, kamu tinggal ikuti instruksi dari website RVM. Kalau kamu instal Ruby pakai RailsInstaller, RVM sudah termasuk dalam instalasi juga.

Kalau pakai RVM, ini beberapa cara install Ruby beda versi.

# Terminal
rvm install ruby-1.9.3 # tidak pakai nomor patch
rvm install ruby-1.9.3-p194 # pakai nomor patch
rvm install ruby-2.0.0

Berikut ini cara untuk memilih versi Ruby yang akan dipakai di Terminal.

# Terminal
rvm use 2.0.0
rvm use 1.9.3-p194
